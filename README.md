# douyin-dl
Command Line Douyin Downloader. Output as original video without watermarks.

# Usage:
```
usage: dydl.py [-o OUTPUT] url

positional arguments:
  url                   URL of Video Page

optional arguments:
  -o OUTPUT, --output OUTPUT
                        Set Output Location
```

# Requirements:

```
python3
requests
urllib
bs4
json
argparse
re
```
